# A-Star-Algorithm-for-Crime-Analytics

#Library Used
matplotlib
numpy
shapefile
time

To run the application, run COMP472_ASSIGNMENT1.py python script.
You will then be prompted to enter the grid size. (Value between 0.01 and 0)
You will also need to enter a crime threshold between 0 and 100.
Next, you will need to enter 4 values, the x and y coordinates of your starting point, followed by the x and y coordinate of your end destination.
Areas in red, represent grids containing a higher crime rate then your input threshold, while blue grids represent areas with a lower number of crime.
The destination is represented as a black point and if an optimal path is found, it is represented as green lines.
